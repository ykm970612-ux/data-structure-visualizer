시각화 프로젝트 설명

Data Structure Visualizer

이 웹사이트는 자료구조 수업에서 배운 Stack, Queue, Linked List, Binary Search Tree의 동작을 사용자가 직접 입력하고 실행하면서 이해할 수 있도록 만든 학습용 시각화 프로젝트입니다.

필수 요구 사항 충족 여부

웹페이지 2개 이상

시각화 페이지와 프로젝트 설명 페이지로 구성했습니다.

이벤트 처리 기능

자료구조 선택, 입력, 삽입, 삭제, 검색, 초기화 버튼 이벤트를 구현했습니다.

반응형 웹 지원

PC, 태블릿, 모바일 화면에 맞춰 카드와 패널 배치가 변경됩니다.

구현 자료구조

Stack

push, pop, top 확인, overflow/underflow 메시지를 지원합니다.

Queue

enqueue, dequeue, front/rear 확인, empty 상태를 시각화합니다.

Linked List

노드 삽입, 삭제, 검색과 포인터 연결 구조를 보여줍니다.

Binary Search Tree

insert, search, inorder traversal과 탐색 경로 하이라이트를 제공합니다.

사용 기술

HTML 페이지 구조와 자료구조 설명 영역 구성
CSS 다크 대시보드 디자인과 반응형 레이아웃
JavaScript 자료구조 연산 로직과 이벤트 처리
D3.js 자료구조 노드 렌더링과 화면 갱신
